    Заполнение базы активов

    Работа начинается с заполнения базы данных активов.

    Вы можете заполнить базу данных активов следующими способами:

    1. Создание активов вручную. Вручную вводятся данные каждого актива, который нужно добавить в базу.

    2. Автоматический сбор данных об активах: при выполнении инвентаризации, из внешних систем через интеграции.

    Рекомендуется использовать комбинацию этих способов: большая часть базы активов заполняется автоматически, единичные активы создаются или редактируются вручную.

    Чтобы настроить сбор данных об активах:

    1. Настройте сбор данных об активах из внешних систем в разделе Настройки → Управление активами → Внешние системы.

    2. Выполните настройку сетей в разделе Активы → Сети. Сети должны быть определены так же, как на сетевом оборудовании организации. Выберите опцию Параметры сканирования → Не сканировать?.

    3. Создайте учетные записи для проведения сканирования в разделе Настройки → Управление активами → Учетные записи.

    4. Запустите сканирование. Вы можете запустить сканирование двумя способами:

    5. Вручную для каждой сети.

    6. Автоматически с помощью политик сканирования. Настройте параметры политик в разделе Настройки → Управление активами → Политики инвентаризации → Политики сканирования.

      Сканирование подсетей, включающих рабочие станции, рекомендуется выполнять в рабочие дни в рабочее время, чтобы сканирование охватило максимальное количество оборудования. Сканирование подсетей, содержащих серверы и служебное оборудование, рекомендуется выполнять в нерабочее время, чтобы не снижать производительность.

    7. После завершения работы политик база активов будет заполнена. Вы можете просмотреть активы в разделе Активы → Оборудование, Активы → Сети, Активы → ПО.

    8. Далее нужно перейти к распределению активов по группам.
